# TypeDuel 🎮⌨️

A fast-paced **multiplayer 2D pixel typing game** built with Godot 4. Race against your opponents by typing faster and more accurately to defeat them in real-time duels!

---

## Features

- **Multiplayer Typing Duels** — Compete against real players in real-time typing battles
- **Online Chat** — Communicate with other players in the lobby and in-game
- **Leaderboards** — See how you rank against the best typists globally
- **Friend List & Invitations** — Add friends and invite them to private duels
- **Custom Rooms** — Create and configure your own game rooms
- **Match History** — Review your past duels and track your progress
- **Account Creation** — Register and save your stats, friends, and history

---

## Requirements

- [Godot 4.2.6](https://godotengine.org/download)
- [Node.js](https://nodejs.org/) (for the game server)

---

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/typeduel.git
```

### 2. Open in Godot

- Launch **Godot 4.2.6**
- Click **Import** and select the cloned project folder
- Open the project

### 3. Start the Server

Open VSCode, navigate to the server directory and start the Node.js server:

```bash
cd server
npm install
npm start
```

### 4. Play the Game

- Go back to Godot
- Press **F5** or click the **Play** button to run the game
- Create an account, find a match, and start typing!

---

## How to Play

1. Log in or create an account
2. Join a public room or create a custom room
3. Wait for an opponent to join
4. When the match starts, type the displayed words as fast and accurately as you can
5. The player who completes the text first wins the duel!
